Part A — Parallel Lines Cut by a Transversal
Problem 1
Given \(m\angle 1 = 65°\), find \(m\angle 2\), \(m\angle 5\), and \(m\angle 8\).
Vertical Angles Corresponding Angles Alternate Exterior Angles
\(\angle 1\) and \(\angle 2\) are a linear pair (supplementary):
\(m\angle 2 = 180° - 65° = \mathbf{115°}\)
\(\angle 1\) and \(\angle 5\) are corresponding angles (equal when lines are parallel):
\(m\angle 5 = 65°\)
\(\angle 1\) and \(\angle 8\) are alternate exterior angles (equal when lines are parallel):
\(m\angle 8 = 65°\)
\(m\angle 2 = 115°\), \(m\angle 5 = 65°\), \(m\angle 8 = 65°\)
Problem 2
\(\angle 4\) and \(\angle 6\): type of angle pair, equal or supplementary, and \(m\angle 6\) if \(m\angle 4 = 112°\).
Consecutive Interior Angles Co-Interior (Same-Side Interior)
(a) \(\angle 4\) and \(\angle 6\) are consecutive (co-interior / same-side interior) angles — they are on the same side of the transversal, between the parallel lines.
(b) They are supplementary: \(m\angle 4 + m\angle 6 = 180°\)
(c) \(m\angle 6 = 180° - 112° = 68°\)
Consecutive interior (co-interior); Supplementary; \(m\angle 6 = 68°\)
Problem 3
Alternate interior angles: \(m\angle 3 = (2x+10)°\) and \(m\angle 5 = (3x-20)°\). Find \(x\) and each angle.
Alternate Interior Angles Solving Equations
Alternate interior angles are equal:
\(2x + 10 = 3x - 20\)
Subtract \(2x\) from both sides: \(10 = x - 20\)
Add 20 to both sides: \(x = 30\)
Check: \(m\angle 3 = 2(30)+10 = 70°\)   \(m\angle 5 = 3(30)-20 = 70°\) ✓
\(x = 30\), \(m\angle 3 = m\angle 5 = 70°\)
Part B — Quadrilaterals
Problem 4
Parallelogram \(ABCD\), \(m\angle A = 72°\). Find \(m\angle B\), \(m\angle C\), \(m\angle D\).
Parallelogram Opposite Angles ≅ Consecutive Angles Supplementary
In a parallelogram, opposite angles are equal: \(m\angle C = m\angle A = 72°\)
Consecutive angles are supplementary: \(m\angle B = 180° - 72° = 108°\)
Opposite: \(m\angle D = m\angle B = 108°\)
Check: \(72 + 108 + 72 + 108 = 360°\) ✓
\(m\angle B = 108°\), \(m\angle C = 72°\), \(m\angle D = 108°\)
Problem 5
Trapezoid \(PQRS\): \(m\angle P = 110°\). Find \(m\angle S\) on the same leg.
Trapezoid Co-Interior Angles
In a trapezoid, angles on the same leg (between the parallel sides) are co-interior (supplementary):
\(m\angle P + m\angle S = 180°\)
\(m\angle S = 180° - 110° = 70°\)
\(m\angle S = 70°\)
Problem 6
4 congruent sides + 4 right angles. Name it and circle all applicable names.
Square Quadrilateral Hierarchy
(a) This is a Square.
(b) A square is also a rectangle (4 right angles ✓), a rhombus (4 ≅ sides ✓), a parallelogram (2 pairs parallel sides ✓).
Trapezoid also applies under the inclusive definition (has ≥ 1 pair of parallel sides).
(a) Square  (b) Parallelogram, Rectangle, Rhombus, Square (and Trapezoid under inclusive def.)
Part C — Polygon Angle Sum Theorem
Problem 7
Interior angle sum of a hexagon (\(n=6\)) using \((n-2)\times 180°\). Draw diagonals from one vertex.
Polygon Angle Sum Triangle Method
\((n-2) \times 180° = (6-2) \times 180° = 4 \times 180°\)
\(= 720°\)
Drawing diagonals from one vertex of the hexagon creates 4 triangles. Each triangle has 180°, so \(4 \times 180° = 720°\). ✓
Interior angle sum \(= 720°\)
Problem 8
Each interior angle of a regular octagon (\(n=8\)).
Regular Polygon Each Interior Angle Formula
Total interior angle sum: \((8-2)\times 180° = 6 \times 180° = 1080°\)
Divide by the number of angles (8):
\(\dfrac{1080°}{8} = 135°\)
Each interior angle \(= 135°\)
Problem 9
Interior angle sum \(= 1260°\). Find \(n\) and name the polygon.
Reverse Angle Sum Solving for n
\((n-2)\times 180 = 1260\)
Divide both sides by 180: \(n - 2 = \dfrac{1260}{180} = 7\)
Add 2: \(n = 9\)
A polygon with 9 sides is a nonagon.
\(n = 9\) — Nonagon
Problem 10
Each exterior angle of a regular pentagon (\(n=5\)). Bonus: each interior angle.
Exterior Angles 360° Rule Interior + Exterior = 180°
All exterior angles of any polygon sum to 360°.
Each exterior angle (regular pentagon): \(\dfrac{360°}{5} = 72°\)
Bonus: Each interior angle \(= 180° - 72° = 108°\)
Confirm using formula: \(\dfrac{(5-2)\times 180°}{5} = \dfrac{540°}{5} = 108°\) ✓
Each exterior angle \(= 72°\); Each interior angle \(= 108°\)
